** 로컬에서 세팅을 완료 후 사용 시 그리드에서 라이센스가 맞지 않다거나 host가 다르다는식의 오류메시지가 출력됩니다.
** 이유는 현재 개발소스에 적용된 rMateGrid가 정식버전(운영서버용)이라서 허용된 IP가 아니기때문에 발생합니다.
** 로컬에서는 평가판(워터마크있음)을 내려받아 사용을 해야 됩니다.
**
** 해결방법은 아래와 같습니다.
** 1. 리아모어소프트에 접속하셔서 DataGrid의 평가판을 내려 받습니다. (http://www.riamore.net/component/dataGrid.html)
** 2. 내려받은 평가판의 압축을 해제합니다.
** 3. 로컬의 /RCMSPT_PT/web/rMateGrid/Component/rMateGridWeb.swf 파일을 Component/rMateGridWeb.swf 파일로 변경합니다.  
** 4. 로컬의 /RCMSPT_PT/web/rMateGrid/LicenseKey/rMateGridLicense.js 파일을 LicenseKey/rMateGridLicense.js 로 변경합니다.
** 5. 로컬의 /RCMSPT_PT/web/rMateGrid/Samples/JS/rMateGridLicense.js 파일을 Samples/JS/rMateGridLicense.js 로 변경합니다. 
      ( 4.0 버전부터는  Samples/JS 에 rMateGridLicense.js 파일이 존재하지 않을 수 있습니다. 
         만약 Samples/JS 폴더에 파일이 존재하지 않아서 수정을 하지 않았는데도 오류가 발생 시 4번의 파일을 5번의 폴더에 복사를 해주면 됩니다. )
** 6. 끝. 테스트.

rMate Grid Flash Version 사용방법

1. index.html 을 클릭하면 다양한 그리드의 샘플화면과, 샘플의 소스와 데이터를 확인 할 수 있습니다.

   rMate Grid를 Local 환경에서 테스트하기 위해서는 다음 작업이 선행되어야 합니다. 
   이는 Adobe Flash Player 보안정책에 해당됩니다.
   (서버에 올리셔서 사용하는 경우에는 아래작업은 불필요합니다.)

   Step1. 아래 해당되는 경로를 따라 폴더로 이동하십시오.

       * Windows XP: Application Data\Macromedia\Flash Player\#Security\FlashPlayerTrust 
         (for example, C:\Documents and Settings\User1\Application Data\Macromedia\Flash Player\#Security\FlashPlayerTrust)
         (Application Data 폴더는 숨겨진 폴더입니다. 
          탐색기 메뉴 중 도구-> 폴더옵션-> 보기-> 숨긴파일 및 폴더 표시를 체크하시거나,
          경로를 직접 입력하시면 해당 폴더가 display 됩니다. 만약 해당 폴더가 없다면, 위의 이름으로 직접 만드시면 됩니다.)

       * Vista : appData\Roaming\Macromedia\FlashPlayer\#Security\FlashPlayerTrust 
      (for example, C:\Users\User1\AppData\Roaming\Macromedia\Flash Player\#Security\FlashPlayerTrust)
          (AppData 폴더는 숨겨진 폴더입니다.
           탐색기 메뉴 중 도구-> 폴더옵션-> 보기-> 숨긴파일 및 폴더 표시를 체크하시거나,
           경로를 직접 입력하시면 해당 폴더가 display 됩니다. 만약 해당 폴더가 없다면, 위의 이름으로 직접 만드시면 됩니다.)

       * Windows 7 : C:\Users\User1\AppData\Roaming\Macromedia\Flash Player\#Security\FlashPlayerTrust
          (AppData 폴더는 숨겨진 폴더입니다.
          탐색기 메뉴 중 도구-> 폴더옵션-> 보기-> 숨긴파일 및 폴더 표시를 체크하시거나,
          경로를 직접 입력하시면 해당 폴더가 display 됩니다. 만약 해당 폴더가 없다면, 위의 이름으로 직접 만드시면 됩니다.)

    ## 위 경로의 User1은 윈도우 시작 시 로그온 한 유저의 이름입니다.


   Step2. FlashPlayerTrust 폴더 안에 rMateTrust.cfg 라는 텍스트 형식의 파일을 만들고,
          메모장을 이용하여 편집하기 위해 여십시오.

   Step3. 제공된 rMate Grid를 복사한 폴더 경로를 rMateTrust.cfg 에 넣어주십시오.
          예를 들어 제공된 샘플의 저장 위치가 C:\Users\User1\Desktop\rMateGrid 라면,
          위 경로를 그대로 rMateTrust.cfg 파일에 추가하십시오.
          (필요에 따라 여러줄이 들어가도 무방하며, 다른 rMate제품의 경로가 들어가 있어도 됩니다)

   Step4. 모든 브라우져를 껐다가 다시 켜시고 index.html을 실행시킵니다.
 

   위 과정은 현재 등록된 폴더 안의 플래시 파일들은 신뢰할 수 있다는 의미입니다.



2. 디렉토리 구조

   디렉토리는 Component, Doc, Samples, Web, LicenseKey 로 되어 있으며, 각각 디렉토리의 역할은 아래와 같습니다.

   가. Component 
       플래시 파일(swf)이 위치하고 있으며, 플래시 파일을 이용하여 모든 그리드를 만들 수 있습니다.
        
   나. Doc
       rMate Grid의 사용방법(메뉴얼)이 위치하고 있습니다.

   다. Samples
       각종 그리드에 대한 예제를 보이기 위하여 필요한 파일들이 위치하고 있습니다. 
       예제를 보기 위해서는 위 1번의 index.html을 click하시면 됩니다.
       예제화면에 나타나는 'VIEW LAYOUT' 과 VIEW DATA' 버튼을 click시에 보여지는 LayoutXml 과 DataXml, 
       그리고 rMate Grid를 사용할 때 필요한 자바스크립트 파일이 위치하고 있습니다.

   라. Web
       위의 샘플이 보여지도록 외곽을 구성하는, 메뉴, 아이콘등의 웹화면을 위해 필요한 html 파일이 위치하고 
       있습니다.     실제 그리드와는 무관한 디렉토리입니다.

   마. LicenseKey
       정품 인증을 위한 라이센스키를 보관해 두는 디렉토리입니다.
       이 디렉토리에 있는 rMateGridLicense.js 파일은 프로그램 개발시에 필요한 디렉토리에 복사하시고,
       Pass를 지정하여 사용하시면 됩니다.   예를들어 본 CD에 있는 샘플 그리드를 실행시키기 위해서는 
       Samples/JS/ 디렉토리에 copy하여 사용하시면 됩니다.(본 CD내에서는 이미 copy하여 두었습니다.)



3. 그리드에 대한 자세한 사용방법은 Doc 드렉토리의 'rMateGrid-사용설명서.pdf' 파일을 참고하시기 바랍니다.
